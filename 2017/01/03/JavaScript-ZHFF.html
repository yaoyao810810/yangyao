数组的转换方法

转换方法

arr.join(字符串)，使用字符串连接数组的每一个元素，返回一个字符串，不改变原数组。

arr.toString() 可以将数组转换为字符串形式，返回字符串，不改变原数组。

数组基本操作方法（均改变原数组）

  • arr.push（item1，tiem2…）可以向数组最后面添加一个或多个元素，返回数组的新长度，改变原数组。
  • arr.pop() 从数组最后删除一个元素，返回删除掉的元素值，改变原数组。
  • arr.unshift（item1，tiem2…）可以向数组前面添加一个或多个元素，返回数组的新长度，改变原数组。需要注意的是，添加多个时，添加的顺序。
  • arr.shift（）从数组前面删除一个元素，返回删除掉的元素值，改变原数组。

位置方法

  • indexOf 和 lastindexOf 这两个方法是 html5 提供的，ie9 以下不支持。
  • arr.indexOf(参数1，参数2) 参数1表示要找到哪个元素，返回找到的索引值，第二个参数是一个索引值，表示从哪个位置开始查找，没有找到，返回-1.

操作方法

  • arr.concat(item1,item2)可以在数组后面连接新的元素，返回新数组，不改变原有数组。
  • 如果传入的参数为数组，将会将数组拆分后进行添加。
  • 复制数组：利用 concat 返回新数组的特性，我们可以在使用时不传参数，达到数组复制的目的。
  • arr.slice(start,end) 可以拷贝数组中从索引 start 开始，到 end 前一位的元素值，以数组形式返回，不改变原数组。
  • arr.splice(start,len,item1,item2…) start 表示截取的起始位置的索引，len 表示截取的元素个数，剩余的参数可选，表示在截取的位置插入新的元素。返回截取的部分(数组形式)，改变原数组。

利用 splice 删除数组指定的元素

排序方法

  • arr.reverse() 翻转数组，返回原数组(一般不使用返回值)，改变原数组。
  • arr.sort() 用于对数组元素进行排序。
  • 数值型数组排序的时候可以接受两个参数
    • function(a-b){return a-b;} 正序
    • function(b-a){return b-a;} 倒序
